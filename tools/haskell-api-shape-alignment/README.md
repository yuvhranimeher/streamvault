# Haskell API Shape Alignment Auditor

Safe read-only StreamVault tool.

It reads:

- live Node API snapshots generated by `haskell-shadow-api-comparator`
- Haskell safe-suite JSON outputs

It writes:

- `out/api-shape-contract.md`
- `out/api-shape-summary.tsv`
- `out/haskell-adjustment-plan.txt`

It does not edit Node server, frontend, playback, catalog, posters, or cache.

Use after the Node server is running and after the shadow comparator has fetched snapshots.
